## 🚀 Quick Start

```bash
# Install globally
npm install -g rafey-shell

# Start chatting
rafey-shell
```

## 💬 Usage

| Command | Description |
|---------|-------------|
| `<message>` | Chat with AI assistant |
| `/help` | Show available commands |
| `/clear` | Clear the terminal |
| `/history` | View recent conversations |
| `/exit` | Exit the shell |
